# homebrew-tap

Homebrew tap for [arthurvasconcelos](https://github.com/arthurvasconcelos) tools.

## Setup

```bash
brew tap arthurvasconcelos/tap
```

## Available formulae

| Formula | Description |
|---------|-------------|
| [overseer](Formula/overseer.rb) | Personal CLI for environment setup and daily automation |

## Install

```bash
brew install arthurvasconcelos/tap/overseer
```

Or, after tapping:

```bash
brew install overseer
```

## Update

```bash
brew upgrade overseer
```
